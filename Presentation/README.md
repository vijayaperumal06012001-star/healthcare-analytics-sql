# Healthcare Analytics with SQL

## Project Overview
This project focuses on analyzing healthcare data using advanced SQL techniques to extract meaningful insights related to patients, doctors, appointments, diagnoses, medications, and billing. The analysis supports data-driven decision-making in healthcare operations.

## Project Objective
The objective of this project is to extract, clean, and analyze healthcare data from multiple relational tables using SQL to generate insights on patients, doctors, diagnoses, medications, and appointments. The goal is to improve operational efficiency and clinical decision-making.

Healthcare Analytics

- Proficient in SQL Joins (INNER, LEFT, RIGHT, FULL OUTER)

## Business Use Cases

### Patient Management
- Analyze patient demographics and engagement
- Identify patients who have never scheduled appointments

### Doctor Performance Analysis
- Measure diagnosis frequency per doctor
- Identify doctors attending the highest number of unique patients

### Appointment Analysis
- Track completed, cancelled, and pending appointments
- Analyze appointment trends

### Medication & Diagnosis Analysis
- Identify commonly prescribed medications
- Analyze prescription duration per diagnosis

### Revenue & Operational Insights
- Link appointment and billing data
- Support healthcare performance monitoring

---

## SQL Analysis Scope
- INNER, LEFT, RIGHT, FULL OUTER Joins
- Aggregate Functions (COUNT, SUM, AVG)
- Window Functions (RANK, DENSE_RANK)
- Conditional Logic using CASE
- Subqueries for complex filtering
- Date-based analysis using DATEDIFF
